# **Техническое задание (ТЗ)**

## **Проект: Реализация веб-блог страницы**

### **1. Цель проекта**

Разработать современную, оптимизированную, кроссбраузерную веб-страницу блог-платформы с использованием технологий Next.js 16 версии и TypeScript. Обеспечить корректную работу на всех устройствах и поисковую оптимизацию контента.

---

### **2. Технологии и инструменты**

* **Next.js 16** (App Router)
* **TypeScript**
* **TailwindCSS**
* **SEO-оптимизация** (метатеги, OG-tags, sitemap, robots)
* **Вёрстка по принципам адаптивного дизайна**
* Поддержка актуальных браузеров:

  * Chrome
  * Firefox
  * Safari
  * Edge

---

### **3. Функциональные требования**

#### **3.1. Структура страницы блога**

* Header
* Статьи
* Footer
* Компонент карточки поста
* SEO-компонент (metadata)

#### **3.2. Функции**

* Отображение списка постов (минимум 6 статей)
* Возможность перехода на страницу поста
* Генерация статических/динамических маршрутов через App Router
* Респонсив адаптация под:

  * 320px (мобильные)
  * 768px (планшеты)
  * 1024px и выше (desktop)
* Применение TailwindCSS для всей вёрстки

---

### **4. Нефункциональные требования**

* Высокая скорость загрузки и оптимизация изображений
* SEO:

  * корректные `title`, `description`
  * OpenGraph разметка
  * генерация sitemap.xml
  * robots.txt
* Адаптивность под все размеры экранов
* Чистая архитектура и читаемый код
* Деплой продакшн версии

---

### **5. Хостинг / Продакшн**

* Разместить проект на любом подходящем хостинге:

  * Vercel (предпочтительно)
  * Netlify
  * Render
  * Другой NextJS-совместимый хостинг
